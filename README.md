This is a small React + Vite project customized for the user's request:
- Welcome modal: "Welcome in the HV Project of Section S" and user name prompt
- Switchable theme (light/dark)
- Track Transport, Electricity, Food, Waste (editable emission factors)
- Chart of monthly totals (Recharts)
- Save data in localStorage
- Export CSV, clear data, reset app

## How to use

1. Download the ZIP and upload the files to your GitHub repo.
2. On Vercel: Import the repository and Deploy (Vite project).
3. Or run locally:
   - `npm install`
   - `npm run dev`
